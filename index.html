Date Translate

Translate WordPress dates with a single line of code.

    Introduction

    This plugin allows you to create insert a hook into your functions file to translate the dates language. It gives you the option of translating days, months and even a shorthand version of the months.

    Download the zip file and install it using your WordPress Dashboard or manually.

    Initializing the Hook

    Create three separate arrays for the days, months and shorthand for months; I will use my native language Albanian to demonstrate functionality.

    Insert the following in your functions.php file.

    $ditet = array(Diele, Hëne, Marte, Mërkure, Enjte, Premte, Shtunë);
$muajt = array(Janar, Shkurt, Mars, Prill, Maj, Qërshor, Korrik, Gusht, Shtator, Tetor, Nëntor, Dhjetor);
$muajt_s = array(Jan, Shk, Mar, Pri, Maj, Qer, Korr, Gusht, Shtat, Tet, Nen, Dhjetor);

qnt_date_translate($ditet, $muajt, $muajt_s);

    Be very careful,

    Week starts with Sunday, ends with Saturday.
